Projetos

Deploy Flask

O projeto "Deploy Flask" é uma aplicação web desenvolvida em Flask, um framework web leve para Python. O objetivo deste projeto é demonstrar o processo de criação e deploy de uma aplicação web simples utilizando Flask, bem como a configuração de um ambiente de produção para garantir que a aplicação esteja disponível para os usuários finais.

Tecnologias Usadas: Flask, Python, HTML e CSS, Git, Render.
